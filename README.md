# Hi, I'm Tingrui (Tony) Zhang

I am an M.Sc. researcher in Electrical and Computer Engineering at McGill University, focusing on AI-enabled wireless communications, full-duplex massive MIMO, hybrid beamforming, and ML/RL-based beam control. My work combines signal processing, measured self-interference channel analysis, and machine learning/reinforcement learning for Tx–Rx isolation enhancement.

## Research interests

- AI/ML for wireless communications
- Full-duplex massive MIMO and self-interference suppression
- Hybrid beamforming and null-steering beamforming
- Signal processing for measured RF/SI channel data
- Reinforcement learning for beam control and wireless optimization

## Technical skills

- **Programming:** Python, MATLAB, PyTorch, NumPy, pandas, scikit-learn
- **Machine Learning:** supervised learning, CNN/DNN training, reinforcement learning, Soft Actor-Critic, model evaluation
- **Wireless / Signal Processing:** full-duplex communications, massive MIMO, hybrid beamforming, null steering, LCMV-based beamforming, measured SI-channel analysis
- **Research Tools:** Jupyter, Google Colab, Git, LaTeX, matplotlib

## Featured projects

- **ML/RL-Based Beamforming for FD mMIMO**  
  Public demo repository in progress. A simplified implementation inspired by my thesis research on nulling-control beamforming, using synthetic data and excluding proprietary measured SI-channel data.

- **MIMO Beamforming Simulation**  
  Planned repository covering ULA steering vectors, beam-pattern visualization, maximum-directivity beamforming, and LCMV-based null-steering examples.

- **Applied Machine Learning Projects**  
  Image classification, medical image segmentation, and time-series forecasting workflows using Python, PyTorch, and scikit-learn.
